Wptobe Signinup is a very light plugin that lets you easily and safely build sign in/up with designed page. 

== Description ==

**The easiest, fastest and most reliable membership plugin**

The wptobe-signinup plugin provides functions for member registration, login and member management required on the website,
It focus on ease of use and fast speed and stability. It offers a Captcha function to protect against spam by default and easily change the design of any theme by selecting a pre-made template without any programming.

Member Registration, Login, Reset Password, Profile, Change Password.
우커머스 쇼핑몰 체크아웃 국내 주소(Daum 우편번호 서비스) 사용 기능.

* The capcha can be used for sign-in/sign-up screen to prevent spam users. 

* Each field can be edited and the position can be easily changed by drag-and-drop. The setting screen and the short code output are the same, so it is very intuitive and easy.

* It provides on/off switch to show or hide each field.

* It is possible to log in by email without username.

* You can set the page to be redirected after sign-in/sign-up.

* No need to put the short code directly in a page, just select it from the selection box.

* You can create different styles and change them whenever you want. For example, create a themeA, themeB C style whenever the theme changes, simply change it in the selection box to change the design. We provides Default/Black/Avada/Enfold design by default.

* It allows users to receive via email if they forget their password. The administrator can modify the content of the email to be sent.

* Members can change their current password to a new password on the profile page.

* It provides an html/css file that outputs sign-in/sign-up/profile independently, so you can easily change and copy it.

* You can reset the plug-in to the initial state at any time.

== Installation ==

1. Change the compressed file name to wptobe-signinup.zip and put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Select pages to display signin, signup and profile.
